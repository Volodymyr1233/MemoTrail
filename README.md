# 📱 MemoTrail

**MemoTrail** is a modern multimedia mobile application designed as an interactive travel diary. It allows users to capture, organize, and explore their travel memories using high-quality media, including support for **8K content**.

Built with the latest Android technologies, MemoTrail combines performance, aesthetics, and usability to deliver a seamless user experience.

---

## ✨ Features

### 📸 Multimedia Management (CRUD)
- Add, view, edit, and delete travel entries
- Support for:
  - Images (including 8K)
  - Videos
  - Voice notes

### 🔍 Search & Filtering
- Search entries by:
  - Title
  - Tags
- Filter and sort by:
  - Date
  - Location

### 🔐 Permissions Handling
- Integration with:
  - Device camera
  - Gallery/media storage

### 🎨 Personalization
- 🌙 Dark Mode support
- 🌐 Multi-language support (i18n):
  - English
  - Polish

### ⚡ Modern UI & Animations
- Built with **Jetpack Compose**
- Reusable UI components
- Smooth animations (e.g., list → detail transitions)

---

## 🏗️ Architecture

The project follows modern Android development standards:

- **Language:** Kotlin
- **Architecture Pattern:** MVVM (Model-View-ViewModel)
- **UI Framework:** Jetpack Compose
- **Database:** Room
- **Navigation:** Jetpack Navigation Component
